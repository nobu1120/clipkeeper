## 詳細な説明（Detailed description）

```
ClipKeep for Notionは、気になったWebページや選択したテキストを、Notionの好きなデータベースへ
確実に保存できるクリッパーです。

■ こんな不便を解消します
・保存したはずのページが実は保存されていなかった（エラーに気づけない）
・保存した後、Notionを開いてタグやカテゴリを手作業で付け直している
・複数のデータベースを使い分けているのに、保存のたびに迷う

■ 主な機能
・ページ全体をワンクリックで抽出（見出し・段落・リスト・引用・コードブロックに整形）
・選択したテキストだけを保存
・保存前にタイトルやタグ（マルチセレクト/セレクトプロパティ）をその場で設定
・保存の成功/失敗をはっきり表示。失敗時はその場で再試行
・複数のNotionデータベースを登録し、保存先を選択
・複数のNotionワークスペース（アカウント）を接続し、使用中のワークスペースを切り替え可能

■ 料金プラン
・無料プラン: 月20クリップまで、データベース1件まで登録可能
・Proプラン: 現在開発中です（クリップ数無制限、複数データベース登録、テンプレート機能などを予定）

■ 必要な準備
Notionの「インテグレーション」機能を使って接続します（Notion公式の設定ページで発行できる
Internal Integration Secretを、本拡張機能のオプション画面に貼り付けるだけです）。
アカウント登録やパスワード入力は必要ありません。

■ プライバシーについて
本拡張機能は独自のサーバーを持ちません。クリップした内容は、あなたのブラウザから直接
Notion公式APIへ送信され、あなた自身のNotionワークスペースに保存されます。
ClipKeep運営者がページ内容や保存先データベースの中身を閲覧・収集することはありません。
詳細はプライバシーポリシーをご確認ください。

■ ご注意
本拡張機能はNotion Labs, Inc.とは無関係の非公式ツールです。
```

## 権限の正当性説明（Permissions justification）

Chrome Web Store の「プライバシーに関する取り組み」タブで、権限ごとに入力する想定の文面です。

| 権限 | 正当性説明（日本語） |
|---|---|
| `contextMenus` | ページ全体または選択したテキストを、右クリックメニューから素早く保存できるようにするために使用します。 |
| `storage` | Notion接続情報（インテグレーショントークン）、今月のクリップ数、登録済みデータベースの一覧をブラウザ内にローカル保存するために使用します。これらの情報は外部サーバーには送信されません。 |
| `activeTab` | ユーザーが拡張機能のアイコンや右クリックメニューを操作した「今開いているタブ」に限定して本文を抽出するために使用します。ユーザーの明示的な操作なしに他のタブやサイトへアクセスすることはありません。 |
| `scripting` | 本文抽出用のスクリプトを、ユーザーが操作した対象タブにのみ注入するために使用します。 |
| `notifications` | 右クリックメニューからのクイック保存時に、保存が成功したか失敗したかをユーザーに通知するために使用します。 |
| `host_permissions`（`https://api.notion.com/*`） | クリップした内容をユーザー自身のNotionワークスペースへ保存するため、Notion公式APIを呼び出す必要があります。この用途以外にこのホストへアクセスすることはありません。 |

`<all_urls>`のような広範なホスト権限は要求していません（`activeTab`方式のため）。
